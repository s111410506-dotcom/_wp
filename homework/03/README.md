# 作業 03 — 個人主頁與 JavaScript

## 說明

### about.html

使用 HTML + Tailwind CSS 實作個人主頁，包含：
- 頂部導覽列（sticky 定位）
- 個人介紹區塊（大頭照、姓名、系級）
- 專業技能卡片（三欄網格）
- 聯絡方式（含圖示連結、email、GitHub、LinkedIn）
- 頁尾版權宣告

採用 **Tailwind CSS CDN** 與 **Font Awesome** 圖示庫，不需額外安裝工具。

### js/hello.js

基礎 JavaScript 練習，在控制台輸出 `hello 你好`。

## 作答方式

直接在瀏覽器開啟 `about.html` 檢視個人主頁效果。`js/hello.js` 可於瀏覽器 DevTools Console 或 Node.js 執行驗證。
