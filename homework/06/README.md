10 題 JavaScript 練習，涵蓋函式、陣列、閉包、非同步等核心概念。

## 題目列表

| # | 主題 | 說明 |
|---|------|------|
| 1 | Higher-Order Function | 傳入 callback 做四則運算 |
| 2 | IIFE 與作用域 | 立即執行函式與封裝 |
| 3 | Array.map | 折扣計算 |
| 4 | 陣列變異 | pop / unshift 修改原陣列 |
| 5 | Currying | 閉包乘法器 |
| 6 | 自訂 filter | 手刻 Array.filter |
| 7 | Array.filter | 物件篩選（成年人） |
| 8 | 傳值 vs 傳參考 | 陣列引用行為分析 |
| 9 | setTimeout | 非同步計時 |
| 10 | reduce + callback | 購物車折扣計算 |
